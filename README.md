# Marketing-Campaign-Analysis---Ashfa-Rofiqoh
End-to-end marketing analysis dashboard to evaluate campaign conversion and customer spending behavior.

## Project Overview
Proyek ini bertujuan untuk menganalisis efektivitas kampanye pemasaran dan memahami perilaku belanja pelanggan (customer spending behavior). Dashboard ini mentransformasi data mentah menjadi wawasan strategis untuk meningkatkan tingkat konversi dan mengoptimalkan alokasi anggaran pemasaran.

## Key Features & Analysis
- **Campaign Evaluation:** Menganalisis tingkat respons pelanggan terhadap promosi (Conversion Rate: 15.05%).
- **Product Performance:** Mengidentifikasi kontributor pendapatan utama, di mana produk **Wine** dan **Meat** menjadi penggerak spending terbesar.
- **Customer Profiling:** Segmentasi pelanggan berdasarkan pendapatan (Average Income), pendidikan, dan jumlah anak di rumah untuk melihat pola loyalitas.
- **Interactive Navigation:** Dilengkapi dengan sidebar menu dan filter (slicer) interaktif untuk eksplorasi data yang lebih mendalam.

## Tech Stack
- **Power BI:** Untuk visualisasi data dan pembuatan dashboard.
- **Power Query:** Untuk proses Data Cleaning dan Transformation (M-Language).
- **DAX (Data Analysis Expressions):** Digunakan untuk membuat Measure seperti Total Revenue, Average Income, dan Response Rate.

## Insights & Recommendations
1. **Targeting High-Spenders:** Fokus kampanye pada segmen pelanggan dengan pendapatan menengah ke atas yang terbukti memiliki spending tinggi di kategori Wine.
2. **Optimizing Response:** Melakukan review pada kelompok pelanggan yang "Rejected" untuk mencari pola penawaran yang lebih relevan di masa mendatang.
